Operational Notes

Keep machine history usable for audits, reporting, and later backend conversion.

  • Every event should show exact machine code, event source, user, and resulting status.
  • Rent machine history should keep vendor and agreement references visible when relevant.
  • PM and breakdown pages can later write into this same history structure without redesign.
  • Allocation and return or replacement flows should remain traceable from one machine timeline.
